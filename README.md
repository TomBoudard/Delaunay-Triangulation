## Files specifications

### Input

`.txt` file containing lines of `x y` coordinates in floats.

Example:
```
1.05 4.3
5.0 2.4
3.2 1.3
1.4 2.8
```

### Output (List of triangles)

`.txt` file containing triplets of indexes for each triangle.

Example:
```
0 1 2
0 1 3
```
